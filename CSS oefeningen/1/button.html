CSS / HTML styling opdracht

Leerdoel

Je leert in deze oefening hoe je een eenvoudige knop maakt en styling toepast binnen CSS. Je leert hoe diverse CSS style eigenschappen zoals randen en kleuren werken. Tevens zul je meer te weten komen over de zogenaamde CSS pseudo klassen.

Opdracht

Style de knop in CSS met de volgende eigenschappen:

  • Geef de border: een dichte lijn, een radius 10 pixels rondom en maak hem zwart van kleur
  • Maak de achtergrond kleur: Rood: 53 / Groen: 126 / Blauw: 189
  • Maak de voorgrond kleur: wit
  • Het font maak je bold, en geef deze een size van 125%
  • Zorg dat de knop oplicht (met een lichtere kleur dan de achtergrondkleur) wanneer je erover heen gaat (hover) met de muis.
  • Bij klikken dient ook de tekst een andere kleur te krijgen (wit wordt bijvoorbeeld zwart)
  • Zoek uit middels CSS styling hoe je voorkomt dat bij klikken de tekst wordt geselecteerd
